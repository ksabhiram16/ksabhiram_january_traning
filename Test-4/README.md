# Supervised Machine Learning Test04

## Problem Statement
Detect spam emails using supervised ML algorithms.

## Dataset
Spam Email Dataset with 1000+ records.

## Preprocessing Steps
- Removed missing values
- Removed duplicates
- Encoded labels
- TF-IDF feature extraction
- Feature scaling
- Train-test split

## Algorithms Used
- Linear Regression
- Decision Tree
- Random Forest
- KNN
- SVM

## Results
Random Forest and SVM achieved highest accuracy.

## Conclusion
SVM performed best in detecting spam emails.
